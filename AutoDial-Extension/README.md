# AutoDial 浏览器插件

一键拨号扩展，点击网页上的手机号即可自动拨打。

## 架构说明

```
┌─────────────┐         WebSocket          ┌─────────────────┐
│   手机端    │ ──────────────────────────→│  Electron PC端   │
│  (主动连接) │                            │  (WebSocket服务器)│
└─────────────┘                            │   监听35432端口  │
                                           └────────┬────────┘
                                                    │
                         WebSocket                   │ 插件连接
                                                    │ (作为客户端)
                                           ┌────────▼────────┐
                                           │   Chrome插件    │
                                           │  (连接PC端)     │
                                           └─────────────────┘
```

**工作流程**：
1. AutoDial PC端启动，创建WebSocket服务器，生成配对码
2. 手机端输入PC的IP和配对码，连接到PC端
3. Chrome插件也连接到PC端（作为第二个客户端）
4. 用户在网页上点击"拨打"按钮
5. 插件发送拨号指令 → PC端收到 → 转发给手机端 → 手机拨号

## 安装步骤

### 1. 加载插件

1. 打开 Chrome，地址栏输入：`chrome://extensions/`
2. 右上角开启 **开发者模式**
3. 点击 **"加载已解压的扩展程序"**
4. 选择 `AutoDial-Extension` 文件夹

### 2. 确保PC端运行

确保 AutoDial PC端已启动并运行。

### 3. 手机连接

1. 打开插件popup（点击工具栏图标）
2. 查看显示的 **本机IP地址** 和 **配对码**
3. 手机端输入IP和配对码进行连接

## 使用方法

1. 确保 PC端已运行，手机已连接
2. 打开任意网页，页面上的手机号右侧会显示金色 **"拨打"** 按钮
3. 点击按钮即可自动拨号

## 功能特性

- 🔍 **自动识别** - 扫描页面手机号并添加按钮
- 📱 **一键拨号** - 点击即可通过手机自动拨打
- 🔄 **自动重连** - 连接断开后自动尝试重连
- 💾 **记忆设置** - 保存常用配置

## 技术说明

- **插件角色**：WebSocket客户端，连接到PC端Electron
- **PC端角色**：WebSocket服务器，管理手机连接和拨号指令
- **手机端角色**：WebSocket客户端，接收拨号指令并执行
